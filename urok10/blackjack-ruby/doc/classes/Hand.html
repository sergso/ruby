Class Hand
In: Hand.rb
Parent: Object

Description

This class represents a single hand of BlackJack.

Synopsis

  hand = Hand.new
  hand.bet = 10
  hand.deal([ Card1, Card2 ])
  puts "BLACKJACK!" if hand.is_bj
  if hand.can_hit
    hand.hit(newcard)
    puts hand.to_s
  end
  hand.stand

new()

Public Instance methods

can_hit()

True if a player can hit on the hand.

can_split()

Returns True if the hand can be split.

deal(cards)

This method adds multiple cards to the hand (usually when dealing).

has_ace()

Returns True if the hand contains an Ace.

has_aces()

Returns True if the hand is a pair of aces.

hit(card)

This method adds a card to the hand.

is_bj()

Returns True if the hand is a BlackJack. Split hands cannot be BlackJack.

is_bust()

True only if hand is bust.

is_done()

Returns True if hand is finished (hand.stand has been called).

is_soft()

Returns True if the hand is soft (i.e. contains an Ace and is < 12).

split()

This method is used to split a hand. It sets the is_split attribute (important for determining BlackJack hands) and returns the new hand.

stand()

Call this method when you want to stand.

to_s()

This method return a string that represents the hand in the form: [Card1|Card2|…]: <Total> <Special Text>; where <Total> can be an "or" statement and <Special Text> can be "BUSTED", "BLACKJACK", etc.

total()

This method returns a total for the hand. IMPORTANT: This method will return a single integer or an integer array. It will return an array of length 2 if the hand is soft (e.g. 7 or 17 for an Ace and a 6). The caller of this method should use the is_soft method to test whether the result will be an Array or not. If the hand is soft, but the attribute stand is True, then the higher value is returned.

total_high()

This method always returns a single integer for the total of the hand. In the case of a soft hand, it will return the higher total.
