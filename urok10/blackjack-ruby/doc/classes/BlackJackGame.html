Class BlackJackGame
In: BlackJackGame.rb
Parent: Object

Description

This class represents the entire BlackJack game. This is not a single round, but the entire game until the user exits. This game flow and calls BlackJackIO for user Input/Output.

Synopsis

  game = BlackJackGame.new
  game.debug = True|False
  game.play_god = True|False
  game.play

Public Instance methods

play()

This method is the meat of the game. It handles all game play until there are no more players or the user quits. 

  Get all input parameters to start game
  While user want to play another shoe
    Create, shuffle and cut shoe
    While the shoe can be played
      Start Round [IO]
      Place Bets
      Deal Round
      Show Hands [IO]
      Play Round
      Settle Round
    end
  end
  Show Game Results [IO]

Private Instance methods

deal_round()

This method deals the initial round of BlackJack, including dealer.

handle_insurance()

This method handles insurance and even money when the dealer‘s up card is an ace.

hit_hand(player, hand)

This method takes care of a player hitting a hand. It deals with the fact that we may in god mode and allow the user to pick the card.

hit_split_hand(player, hand)

This method takes card of hitting a split hand. It deals with the rules of split hands:

  1. You can only split so many hands, determined by player.has_max_hands
  2. When you split aces, you only get one more card, unless it too is an ace
init_game()

This method‘s job is to:

  1. Display welcome msg
  2. Get number of decks
  3. Get number of players
  4. Get player names
  5. Create the player objects, add them to the players array
  6. Create the Dealer object and add him to the players array
need_bet_change()

This method returns true if at least one player needs to change their bet because their bet exceeds their bankroll.

place_your_bets()

This method handles players placing their bets and creates the Hand.

play_round()

This method handles playing out the round of BlackJack.

settle_round()

This method takes care of paying players and taking their money.
