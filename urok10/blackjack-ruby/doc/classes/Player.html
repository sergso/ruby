Class Player
In: Player.rb
Parent: Object

Description

This class represnts a BlackJack player, complete with their name, bankroll, bet increment and hand.

Synopsis

  player = Player.new("Bob", 1000)
  player.bet = 100 # High roller
  puts "#{player.name}, you have #{player.bankroll} bucks."
  if hand.has_aces
    player.split_hand(hand) # or do you like to lose money?
  ...

Public Instance methods

all_bankrolls()

This method returns the total of everyone‘s bankroll

bankroll_to_s()

This method returns a string in the form: Name [Bankroll].

can_afford_insurance()

This method should be used to determine if the player can put up half their bet for insurance.

can_double_bet()

This method should be used to determine if the player can double their bet. Used for splits, double down or any other purpose.

can_split_hand(hand)

This method returns true if the hand can be split and the player has not exceeded the maximum number of hands.

double_bet(hand)

This method takes care of doubling the player‘s bet.

finish_round()

This method destroys a player‘s hand.

hand_to_s(hand)

This method returns a string for the hand in the form: Name [HandIndex] [HandBet] [Bankroll].

has_a_hand()

This method returns True if the player has at least one hand

has_max_hands()

This method returns True if the player has the maximum number of hands.

has_multiple_hands()

This method returns True if the Player has multiple hands, nil otherwise.

is_dealer()

Returns True if the Player is the Dealer.

lose_insurance_bet(hand)
place_bet(hand)

This method places the player‘s bet on the hand.

place_insurance_bet(hand, insurance_bet)

This method places an insurance bet on the hand.

split_hand(hand)

This method should be used when the player splits their hand. Send in the hand that should be split and this method will create and return the new hand.

start_hand()

This method places a bet for the player. It instatiates the player‘s hand and deducts the bet from the player‘s bankroll. It returns the hand.

summary()

This method returns a summary for the player in the form: Name [Hands Played] [Bankroll]

to_s()

This method returns a string in the form: Name [Bet] [Bankroll].

total_hands()

Returns the total number of hands dealt.

win_bet(winnings)

This method adds the winnings to the player‘s bankroll.

win_insurance_bet(hand, winnings)

This method is called when the player wins the insurance bet.
