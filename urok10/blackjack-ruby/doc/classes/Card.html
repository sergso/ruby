Class Card
In: Card.rb
Parent: Object

Description

This class represents a single card and consists of 3 elements: symbol, suit and value. The symbol is one character: 2..9, [T]en, [A]ce, [J]ack, etc. The suit is one character: [S]pade, [H]eart, [C]lub or [D]iamond. Note: The value of an [A]ce is set at 1. It is up to the caller to handle other cases like "soft" BlackJack hands.

  puts card.to_s
  if card.is_ace
