Class CardSuits
In: CardSuits.rb
Parent: Array

Description

This class is a simple array that represents a card suit. The suit is one character in length and the array is ordered from lowest to highest suit rank.

  • [0] D => Diamonds
  • [1] C => Clubs
  • [2] H => Hearts
  • [3] S => Spades

Synopsis

  suits = CardSuits.new
  suits.each { |item|block| }
  for suit in suits
    puts suit
  end
