Class Deck
In: Deck.rb
Parent: Object

Description

This class represents a deck of 52 cards; jokers excluded. It uses the CardSuits and CardSymbols classes to build a deck of Card objects. Cards should not be taken from or put into this object; inherit from this class if you need to do that.

Synopsis

  deck = Deck.new
  deck.shuffle
  deck.cut(34)
  puts deck.to_s
  for card in deck.cards
    puts card.to_s
  end

new()

Public Instance methods

cut(cut_at)

Given a card number, cuts the deck. Must be 1..{MAX_CARD - 1} to make sure we get in between 2 cards.

shuffle()

Shuffles the deck.

to_s()

This method returns the whole deck as a string. It includes an index (1 - based) and Card ID with spaces in between each card; no line break at the end.
